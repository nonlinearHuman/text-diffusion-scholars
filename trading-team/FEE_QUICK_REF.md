```
交易金额    双边手续费    保本涨幅
100元       ~10%         需涨10%+
300元       ~3.3%        需涨3.3%+
500元       ~2.0%        需涨2.0%+

```

## 策略调整

### 买入信号
1. **动量突破**：涨幅>2% + 量比>1.2
   - 建议金额：≥500元（降低手续费占比）

2. **超跌反弹**：跌幅>3% + 量比>1.0
   - 建议金额：≥500元

### 卖出规则
1. **止损**：亏损5%（实际约6%含手续费）
2. **止盈**：盈利8%（实际约6.8%扣手续费）
3. **保本止损**（新增）：
   - 持仓≥3天
   - 盈利<2%
   - 建议退出避免手续费亏损

### 仓位管理
- 单只最大仓位：40%
- 最小交易金额：300元（建议500元+）
- 保留现金：≥10%

### 查看手续费统计
```bash
cat investment_logs/state.json | grep total_fees_paid
```

## 实战建议

### ✅ 推荐
1. 单笔交易≥500元
2. 持仓至少3天（降低交易频率）
3. 止盈目标≥8%（覆盖手续费）
4. 使用系统计算的真实收益

### ❌ 避免
1. 小额频繁交易（手续费吃利润）
2. 微利就卖（可能实际亏损）
3. 不看手续费统计
