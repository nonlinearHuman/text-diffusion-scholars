### 1. 手续费成本 ✅
```
印花税：卖出0.1%
```

### 2. T+1交易限制 ✅
```
规则：当日买入的股票，次日才能卖出
实现：卖出前检查买入日期
影响：无法日内止损，风险暴露增加
```
**测试结果：**
```
买入平安银行 → 尝试当日卖出 → ❌ T+1限制：今日买入，无法卖出
```

---

### 3. 最小交易单位 ✅
```
规则：A股最小买入100股（1手）
实现：买入前检查股数是否≥100
影响：小资金无法精确控制仓位
```
**测试结果：**
```
尝试买入50股 → ❌ 最小交易单位：需要至少买入100股
```

---

### 4. 滑点成本 ✅
```
规则：实际成交价≠预期价格
实现：买入价上浮0.2%，卖出价下浮0.2%
影响：双边滑点约0.4%，增加交易成本
```
**测试结果：**
```
目标价：10.00元
买入价：10.02元（+0.2%滑点）
卖出价：10.479元（-0.2%滑点，目标10.5元）
双边滑点：4.10元
```

---

### 5. 涨跌停限制 ✅
```
规则：涨停无法买入，跌停需谨慎
实现：涨停股票自动过滤
影响：可能错过强势股，但避免追高风险
```

---

### 6. 流动性筛选 ✅
```
规则：成交额≥1000万
实现：过滤低成交量股票
影响：避免买入困难、滑点过大
```

## 📊 完整成本分析

### 示例：1000元交易平安银行

#### 理想情况（表面）
```
买入：100股 @ 10.00元 = 1000元
卖出：100股 @ 10.50元 = 1050元
收益：+50元（5%）
```

#### 实际情况（考虑所有因素）
```
买入：
  目标价：10.00元
  滑点价：10.02元（+0.2%）
  交易金额：1002元
  手续费：5元
  总成本：1007元

卖出（涨5%）：
  目标价：10.50元
  滑点价：10.479元（-0.2%）
  交易金额：1047.9元
  手续费：6.05元
  实际到账：1041.85元

净利润：1041.85 - 1007 = +34.85元
实际收益率：3.47%（而非5%）
总成本：手续费11.05元 + 滑点4.10元 = 15.15元
```

**关键发现：** 表面5%收益，实际仅3.47%，成本吞噬1.53%！
