## 二、扩散模型技术深度解析

### 2.1 数学基础

#### 2.1.1 概率论框架

扩散模型本质是**变分推断(Variational Inference)** 的近似：

```
p(x₀) = ∫ p(x₀:xT) dx₁:xT

其中 p(x₀:xT) = p(xT) ∏ₜ pθ(xₜ₋₁|xₜ)
```

**关键假设**：
- 前向过程 q(xₜ|xₜ₋₁) 是已知的高斯过程
- 逆向过程 pθ(xₜ₋₁|xₜ) 是学习的高斯过程

#### 2.1.2 前向扩散过程

```python
# 前向过程：逐步添加噪声
for t in range(T):
    βₜ = β₁ + (βT - β₁) * t / T  # 噪声调度
    ε ~ N(0, I)
    xₜ = √(1-βₜ) * xₜ₋₁ + √βₜ * ε
```

**性质**：
- 任意时刻 xₜ 可以直接由 x₀ 和 ε 计算：
```python
xₜ = √(ᾱₜ) * x₀ + √(1-ᾱₜ) * ε  # ᾱₜ = ∏(1-βₛ)
```

#### 2.1.3 逆向扩散过程

学习目标：**预测噪声 ε**

```python
# 简化训练目标
L = ||ε - εθ(xₜ, t, c)||²

# 其中 εθ 是神经网络预测的噪声
# xₜ: 当前带噪图像
# t: 时间步 (embedding)
# c: 条件信息 (文本、图像等)
```

#### 2.1.4 采样过程

```python
# DDPM 采样 (1000步)
xT ~ N(0, I)
for t in reversed(range(T)):
    z ~ N(0, I) if t > 0 else 0
    xₜ₋₁ = (xₜ - √(1-αₜ) * εθ(xₜ,t)/√αₜ) / √αₜ + σₜ * z

# DDIM 采样 (50步，更快)
for t in sequence:
    xₜ₋₁ = √αₜ₋₁ * (xₜ/√ᾱₜ - √(1-ᾱₜ)/√ᾱₜ * εθ) + √(1-ᾱₜ₋₁) * εθ
```

### 2.2 主流扩散模型架构

#### 2.2.1 DDPM (Denoising Diffusion Probabilistic Models)

**提出**：2020, Ho et al., Google Research

**特点**：
- 原始扩散框架
- U-Net作为噪声预测网络
- 1000步采样，速度慢

**核心代码逻辑**：
```python
class DiffusionModel(nn.Module):
    def forward(self, x, t, cond):
        # Time embedding
        t_emb = self.time_mlp(sinusoidal_embedding(t))
        
        # U-Net forward with conditioning
        return self.unet(x, t_emb, cond)
    
    def training_step(self, x0):
        # 随机采样时间步
        t = torch.randint(0, T, (B,))
        # 随机噪声
        ε = torch.randn_like(x0)
        # 前向加噪
        xₜ = sqrt(alpha_bar[t]) * x0 + sqrt(1-alpha_bar[t]) * ε
        # 预测噪声
        ε_pred = self(xₜ, t)
        # 损失
        return MSE(ε, ε_pred)
```

#### 2.2.2 LDM (Latent Diffusion Model)

**提出**：2021, Rombach et al., CompVis

**核心创新**：在**潜在空间(Latent Space)**中进行扩散

```
原始图像 (512×512×3)
    ↓ VAE Encoder
潜在表示 (64×64×4)  # 压缩8倍
    ↓ 扩散+去噪
    ↓
生成潜在表示
    ↓ VAE Decoder
输出图像
```

**优势**：
- 计算量减少 ~64倍
- 质量不下降
- 速度大幅提升

**Stable Diffusion 架构**：
```
文本 → CLIP Text Encoder → 条件向量
                ↓
潜在空间 → UNet (交叉注意力注入条件) → 去噪
                ↓
            VAE Decoder → 图像
```

#### 2.2.4 DiT (Diffusion Transformer)

**提出**：2023, Peebles & Xie, NYU/Stanford

**核心创新**：用Transformer替换U-Net

```python
# DiT Block
class DiTBlock(nn.Module):
    def __init__(self, dim, n_heads):
        self.norm1 = nn.LayerNorm(dim)
        self.attn = nn.MultiheadAttention(dim, n_heads)
        self.norm2 = nn.LayerNorm(dim)
        self.cross_attn = nn.MultiheadAttention(dim, n_heads)
        self.mlp = nn.Sequential(
            nn.Linear(dim, dim * 4),
            nn.GELU(),
            nn.Linear(dim * 4, dim)
        )
    
    def forward(self, x, cond):
        # 自注意力
        x = x + self.attn(self.norm1(x), x, x)[0]
        # 交叉注意力 (注入条件)
        x = x + self.cross_attn(self.norm2(x), cond, cond)[0]
        # FFN
        x = x + self.mlp(x)
        return x
```

**DiT vs U-Net 对比**：

| 维度 | U-Net | DiT |
|------|-------|-----|
| 参数量 | ~1B (SD) | 可扩展至10B+ |
| 架构 | 编码-解码 | 纯Transformer |
| 注意力 | 局部+全局 | 全局自注意力 |
| 扩展性 | 一般 | 优秀 |
| 训练稳定性 | 较稳定 | 需调节 |
| 推理速度 | 快 | 较慢 |

#### 2.2.5 近期重要变体

**1. Rectified Flow (RF)**

```python
# 核心思想：直线插值而非噪声
# 传统扩散：x₁ = x₀ + ε (随机路径)
# Rectified Flow：x₁ = x₀ + v (确定路径)

# 训练目标
minimize ||v - vθ(x, t)||²
# 其中 v = x₁ - x₀ (流向量)
```

**优势**：采样只需4-10步

**代表工作**：
- **InstantID** (2024)：人脸一致性
- **FLUX** (2024)：Stability AI新作

**2. Consistency Models (CM)**

```python
# 核心思想：直接预测 x₀ 而非 ε
# 性质：f(xₜ, t) = f(xₜ', t') 当 xₜ ≈ xₜ'

# 训练目标（无对抗）
L = ||x₀ - F(xₜ, t)||²
# 其中 F 是 consistency function
```

**代表工作**：OpenAI **LCM (Latent Consistency Model)**

**3. Flow Matching**

```python
# 统一框架：扩散 ≈ 常微分方程
# 核心：学习向量场 vₜ(x)

# 训练目标
L = ||vₜ(xₜ) - vθ(xₜ, t)||²
```

```python
# DPM-Solver
# 使用高阶ODE求解器，减少步数

# 核心思想：
# xₜ₋₁ = xₜ + (tₜ₋₁ - tₜ) * f(xₜ, t) + ...
```

**3. 噪声调度**

```python
# Cosine schedule (更平滑)
βₜ = f(t/T) / f(0)

# Linear schedule (原始)
βₜ = β₁ + (βT - β₁) * t/T
```

### 2.4 条件控制技术

#### 2.4.1 Classifier-Free Guidance (CFG)

**核心公式**：
```python
εθ(xₜ, c) = εθ(xₜ, ∅) + w * (εθ(xₜ, c) - εθ(xₜ, ∅))
# w: guidance scale (通常 5-15)
# c: 条件; ∅: 空条件 (unconditional)
```

**作用**：增强条件控制效果

**原理**：
- w=1: 无引导
- w>1: 强调条件，抑制随机性
- w过大: 图像质量下降

#### 2.4.2 ControlNet

**架构**：
```
输入图像 → 编码器分支 → 零卷积 → 注入主网络
                ↓
          可训练副本
```

**训练策略**：
1. 锁定主UNet
2. 训练ControlNet分支
3. 条件图像→边缘/姿态/深度图

**支持的控制类型**：
- Canny边缘
- 深度图 (Midas)
- 姿态 (OpenPose)
- 线稿 (Scribble)
- 法线 (Normal Map)

#### 2.4.3 IP-Adapter

**核心思想**：解耦图像Prompt和文本Prompt

```python
# 架构
图像 → 图像Encoder → 图像特征
                    ↓
文本 → 文本Encoder → 文本特征
                    ↓
           交叉注意力融合
                    ↓
               UNet
```

**优势**：
- 图像控制不干扰文本控制
- 可同时使用多个参考图

#### 2.4.4 T2I-Adapter

**特点**：
- 轻量级 (~75M参数)
- 冻结主模型
- 预测指导特征

#### 2.6.2 LoRA 微调

```python
# 低秩适配
# 冻结原始权重，添加可训练旁路

class LoRA(nn.Module):
    def __init__(self, in_dim, out_dim, rank=8):
        self.lora_A = nn.Parameter(torch.randn(rank, in_dim))
        self.lora_B = nn.Parameter(torch.randn(out_dim, rank))
    
    def forward(self, x):
        return (self.lora_B @ self.lora_A) @ x
```

**优势**：
- 训练参数量减少90%+
- 快速微调 (~1小时)
- 可组合多个LoRA
