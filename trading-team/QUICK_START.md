# 量化投资实战系统

## 目标设定
- **初始资金**: 1000元
- **月收益目标**: 10%
- **日均收益目标**: 约0.43%
- **风险等级**: 中等

## 使用流程

### 方式1：每日运行获取信号
```bash
cd /Users/gesong/.openclaw/workspace/trading-team
./start_investing.sh
```

系统会输出：
- 今日交易信号
- 建议买入/卖出股票
- 建议仓位金额
- 风险提示

### 方式2：记录交易
执行交易后，记录到系统：
```bash
python quant_investor.py
# 然后输入：
# BUY 000001 平安银行 10.5 100
# 或
# SELL 000001 平安银行 11.0 100
```

### 方式3：查看收益
```bash
cat investment_logs/state.json
```

## 当前策略

### 买入策略
1. **动量突破**：涨幅>2% + 量比>1.2
2. **超跌反弹**：跌幅>3% + 量比>1.0

### 卖出策略
1. **止损**：亏损>5% 强制卖出
2. **止盈**：盈利>8% 建议卖出

### 仓位管理
- 单只股票最大仓位：40%
- 弱信号减仓：24%
- 保留现金：至少10%

## 实战注意事项

### ⚠️ 网络问题
当前akshare无法连接东方财富API（代理问题）

**解决方案：**
1. 关闭系统代理
2. 或配置Python代理：
   ```python
   import os
   os.environ['HTTP_PROXY'] = 'http://your-proxy:port'
   os.environ['HTTPS_PROXY'] = 'http://your-proxy:port'
   ```
3. 或使用手机热点（无代理环境）

### 💡 实盘建议
1. **先观察**：运行3-5天，观察信号质量
2. **小仓位试水**：先用100-200元测试
3. **记录复盘**：每笔交易记录理由，周末复盘
4. **严格止损**：亏损5%必须卖出

### 📊 收益跟踪
系统会记录：
- 总资产变化
- 每日收益
- 月目标进度
- 胜率统计
