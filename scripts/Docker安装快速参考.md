## 📋 安装步骤

1. **下载**：点击镜像站点的 `Docker.dmg` 文件
2. **安装**：双击 dmg → 拖到 Applications
3. **启动**：打开 Docker Desktop 应用
4. **验证**：终端运行 `docker --version`

## 🆘 常见问题

### Q: 镜像站点打不开？
A: 尝试另一个镜像站点，或告诉我提供其他方案

### Q: 下载速度慢？
A: 国内镜像通常很快，如果慢可以尝试：
   - 更换时间段下载
   - 使用下载工具（如 aria2）
   - 告诉我，我提供其他方案

### Q: 安装后打不开？
A: 检查：
   - macOS 版本是否 ≥ 12
   - 是否是 Apple Silicon 版本
   - 安全设置是否允许第三方应用
