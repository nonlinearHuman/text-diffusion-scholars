## 决策记录

1. **关卡设计调整**：
   - 原设计：3关（教学 → 简单 → 地狱）
   - 新设计：2关（教学 → 地狱）
   - 理由：简化开发，更快验证市场

2. **命名选择**：
   - 方块大师（长期运营）→ 平缓难度，口碑传播
   - 方块别方（爆款收割）→ 地狱难度，病毒传播

3. **技术选型**：
   - 微信原生小程序（性能最优）
   - 微信云开发（免费额度）
   - 无框架开销

4. **商业模式**：
   - 方块大师：广告 + 内购（长期）
   - 方块别方：广告为主（快速收割）

5. **开发周期**：
   - 方块别方：2周快速开发（03-08 ~ 03-22）
   - MVP范围：2关 + 分享复活 + 激励视频广告

6. **难度策略**：
   - 第2关简单形状概率降低到15%
   - 困难形状概率提高到55%
   - 每个格子颜色随机
   - 预置8个随机方块
