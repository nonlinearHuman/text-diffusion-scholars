## 二、技术架构深度解析

### 2.1 扩散模型基础原理

#### 2.1.1 前向扩散过程 (Forward Diffusion)
```
x₀ → x₁ → x₂ → ... → xT (T=1000步)
```
- 逐步添加高斯噪声，最终接近标准正态分布
- 关键公式：q(xₜ|xₜ₋₁) = N(xₜ; √(1-βₜ)xₜ₋₁, βₜI)

#### 2.1.2 逆向过程 (Reverse Process)
```
xT → xT₋₁ → ... → x₀
```
- 学习去噪：p(xₜ₋₁|xₜ) = N(xₜ₋₁; μₜ(xₜ), Σₜ(xₜ))
- 核心：预测噪声 ε 的神经网络 (U-Net → Transformer)

#### 2.1.3 训练目标
```
minimize: ||ε - εθ(xₜ, t, c)||
```
其中 c 为条件信息（文本、图像、布局等）

### 2.2 扩散+LLM融合的三条技术路线

#### 路线1：自回归扩散 (Autoregressive Diffusion)

**代表工作**：
- **AR-Diffusion** (Google)
- **Pixel Diffusion** (Google)
- **Nano Banana 2** (Gemini 3.1 Flash Image)

**核心思想**：将扩散过程建模为token-by-token生成

```
传统扩散：一次性生成完整图像
自回归扩散：逐块/逐行生成，每块依赖前序内容
```

**技术优势**：
- ✅ 更好集成LLM的语义理解能力
- ✅ 支持更精确的局部控制
- ✅ 便于实现内容一致性

**架构图**：
```
文本输入 → LLM Encoder → 语义向量 → Diffusion Decoder → 图像输出
                                      ↓
                              自回归生成策略
```

#### 路线2：Transformer扩散 (DiT - Diffusion Transformer)

**代表工作**：
- **Sora** (OpenAI)
- **DiT** (NYU/Stanford)
- **Runway Gen-3**

**核心创新**：用Transformer替换传统U-Net

| 组件 | U-Net (传统) | Transformer (DiT) |
|------|-------------|------------------|
| 架构 | 编码器-解码器 | 标准ViT |
| 注意力 | 局部+全局 | 全局自注意力 |
| 参数量 | ~1B | 可扩展至10B+ |
| 推理速度 | 较快 | 较慢 |
| 扩展性 | 有限 | 优秀 |

**DiT关键组件**：
1. **Patchify**：将图像划分为 2×2 或 4×4 的patches
2. **In-context Conditioning**：注入 timestep 和 condition
3. **Adaptive Layer Norm (AdaLN)**：动态归一化参数

#### 路线3：端到端多模态融合

**代表工作**：
- **Gemini 3** (Google)
- **GPT-5** (OpenAI)
- **Qwen2.5-VL** (阿里)

**架构特点**：
- 单一Transformer同时处理文本、图像、视频、音频
- 统一tokenizer：图像/视频 → discrete tokens
- 扩散解码作为生成末梢

### 2.3 关键技术创新 (2025-2026)

#### 2.3.1 少步数采样 (Few-step Sampling)

| 方法 | 步数 | 质量损失 | 代表工作 |
|------|------|---------|---------|
| DDPM | 1000 | 0% | 原始论文 |
| DDIM | 50 | ~5% | 2020 |
| LCM (Latent Consistency Model) | 4-8 | ~10% | 2024 |
| SDXL-Lightning | 2-4 | ~15% | 2025 |

**LCM核心技术**：
- 直接预测 x₀ 而非 ε
-  Consistency属性：f(xₜ, t) ≈ f(xₜ', t') 当 xₜ ≈ xₜ'

#### 2.3.2 条件控制增强

| 技术 | 功能 | 代表工作 |
|------|------|---------|
| ControlNet | 边缘/姿态/深度控制 | 2023 |
| T2I-Adapter | 轻量级控制 | 2024 |
| IP-Adapter | 参考图控制 | 2024 |
| Composer | 多条件组合 | 2024 |

#### 2.3.3 知识蒸馏

**学生-教师架构**：
```
教师模型 (10B参数, 50步) → 知识 → 学生模型 (500M参数, 4步)
```

**方法对比**：
| 蒸馏方法 | 加速比 | 质量保持 |
|----------|--------|----------|
| 渐进式蒸馏 | 8x | 95% |
| 对抗性蒸馏 | 10x | 90% |
| 一致性蒸馏 (LCM) | 20x | 85% |
