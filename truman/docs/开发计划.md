- **LLM**: OpenClaw API (GLM-4-Flash)

### 1. Discord Bot 代理配置
**问题**: Discord 在某些地区需要代理
**解决方案**: 
- 在 .env 中配置 HTTPS_PROXY
- 代码已支持 WebSocket + HTTP 双重代理

### 2. LLM 集成
**问题**: 需要调用 LLM 生成自然语言
**解决方案**:
- 使用 OpenClaw API
- 已创建 utils/llm.js 工具
- 配置 GLM-4-Flash 模型

### 3. 数据库选择
**问题**: 需要稳定可靠的云数据库
**解决方案**:
- 使用 Neon PostgreSQL
- 免费 tier 足够开发使用
- 自动备份和扩展
