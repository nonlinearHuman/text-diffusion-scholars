# 🤝 人机协作交易协议

## 协作模式定义

### 我的职责
1. **主动通知**：在交易时点通知你
2. **具体指令**：告知买入/卖出哪只股票、价格、股数
3. **策略调整**：根据你的反馈动态调整
4. **风险监控**：监控本金保护线

### 你的职责
1. **执行反馈**：告知是否买卖成功
2. **账户状态**：告知当前账户价值
3. **最终决策**：可以拒绝我的建议

---

## 📢 沟通格式

### 我的交易通知格式
```
🔔 交易信号 - [日期 时间]

【操作】买入/卖出
【股票】平安银行(000001)
【价格】10.01元（卖一价）
【股数】100股
【金额】1001元
【策略】动量突破
【理由】涨幅3.2%，量比1.5

⚠️ 风险提示：
- 保本涨幅：2.8%
- 止损价：9.70元（-3%）
- 止盈价：10.50元（+5%）
- 建议保留现金：799元（20%）

请确认是否执行？
```

### 你的反馈格式
```
✅ 执行成功 / ❌ 未执行

【账户状态】
- 现金：XXX元
- 持仓：XXX元
- 总资产：XXX元
- 盈亏：XXX元
```

## ⏰ 通知时机

### 交易时段
```
早盘：9:30-11:30
午盘：13:00-15:00

通知时间点：
- 开盘后（9:35）：市场稳定，给出当日信号
- 盘中（10:30）：检查是否需要调整
- 尾盘（14:50）：是否需要尾盘操作
```

### 非交易时段
```
- 每日收盘后：当日总结
- 重大风险：随时通知
```

## 🎯 决策流程

### 买入流程
```
1. 我发现买入信号
2. 计算仓位和风险
3. 通知你具体指令
4. 等待你的执行反馈
5. 更新账户状态
6. 监控持仓风险
```

### 卖出流程
```
1. 触发止损/止盈条件
2. 计算卖出价格
3. 通知你具体指令
4. 等待你的执行反馈
5. 更新账户状态
6. 记录交易结果
```

---

## ⚠️ 风险控制

### 本金保护机制
```
我会在通知中标注：
- 当前总资产
- 距离预警线距离
- 风险等级

如果触发预警线：
- 我会暂停所有买入建议
- 建议持有现金观望
```

## 🚨 特殊情况处理

### 情况1：你拒绝执行
```
我：建议买入平安银行...
你：❌ 不执行，我觉得风险太大

我：✅ 已记录，等待下一个机会
    当前现金：1000元
    状态：观望
```

### 情况2：执行失败
```
我：建议买入...
你：❌ 涨停了买不进去

我：✅ 已记录，调整策略
    标记该股票涨停，跳过
    寻找下一个机会
```

### 情况3：账户异常
```
你：总资产只有900元了

我：🚨 本金亏损！触发熔断线！
    立即停止所有交易建议
    请确认是否继续？
```
