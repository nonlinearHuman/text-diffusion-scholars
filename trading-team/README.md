# A股交易团队系统

## 系统已启动 ✅

### 当前状态
- **初始资金**: 1000元
- **目标收益**: 10000元（10倍）
- **剩余资金**: 700元
- **持仓**: 平安银行(000001) - 300元
- **策略**: 超跌反弹

### 团队成员
1. **数据分析师** - 市场数据收集（akshare实时数据）
2. **策略研究员** - 多策略分析（打板/龙头/超跌/资金博弈）
3. **风险控制官** - 仓位管理（单只最大30%）
4. **交易执行员** - 执行交易并记录

### 策略池
- **打板策略**: 追踪涨停板股票，捕捉连板机会
- **龙头战法**: 板块轮动，捕捉龙头切换
- **超跌反弹**: 短期超跌标的，博反弹
- **资金博弈**: 追踪主力资金流向

### 风险控制规则
- 单只股票最大仓位：30%
- 单日最大亏损：5%
- 连续止损3次：暂停交易
